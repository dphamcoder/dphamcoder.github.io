## Nội dung chính của Website
Trang web được xây dựng bằng HTML/CSS tĩnh, bao gồm các phần:
- **Giới thiệu:** Thông tin cơ bản và định hướng nghề nghiệp (IT Helpdesk -> System Admin).
- **Dự án:** - Triển khai hạ tầng mạng thực tế (Cisco, DrayTek, Mikrotik, VLAN).
  - Web xem phim trực tuyến NGHENPHIM.COM.
  - Xây dựng và quản lý kênh YouTube đánh giá phần cứng công nghệ.
  - Các bài Lab mô phỏng mạng CCNA trên GNS3/Packet Tracer.
- **Chứng chỉ:** Liệt kê các chứng nhận chuyên môn (CCNA, Mendix, Fortinet).
- **Liên hệ:** Kênh kết nối qua Email và YouTube.
